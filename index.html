VAFA — Voice-Activated Form Assistant for Banking

Multilingual speech-to-structured-data pipeline for banking forms.

Designed and deployed a multilingual NLP pipeline that converts spoken Hindi/English input into structured banking form fields. DistilBERT classifies utterances into entities like Name or Amount, followed by a regex engine that extracts values such as 10-digit phone numbers or monetary amounts. Deployed with Streamlit and Web Speech API, enabling real-time usage for 500+ beta users with F1 > 0.98.
